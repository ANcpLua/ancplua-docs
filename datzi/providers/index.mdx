# Model Providers

Datzi can use many LLM providers. Pick a provider, authenticate, then set the
default model as `provider/model`.

Looking for chat channel docs (WhatsApp/Telegram/Discord/Slack/Mattermost (plugin)/etc.)? See [Channels](/channels).

## Highlight: Venice (Venice AI)

Venice is our recommended Venice AI setup for privacy-first inference with an option to use Opus for hard tasks.

- Default: `venice/llama-3.3-70b`
- Best overall: `venice/llama-3.3-70b`

See [Venice AI](/providers/venice).

## Quick start

1. Authenticate with the provider (usually via `datzi onboard`).
2. Set the default model:

```json5
{
  agents: {
    defaults: {
      model: {
        primary: 'ollama/qwen3-coder:32b'
      }
    }
  }
}
```
