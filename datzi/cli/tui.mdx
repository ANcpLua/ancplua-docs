# `datzi tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
datzi tui
datzi tui --url ws://127.0.0.1:18789 --token <token>
datzi tui --session main --deliver
```
