# `datzi setup`

Initialize `~/.datzi/datzi.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
datzi setup
datzi setup --workspace ~/.datzi/workspace
```

To run the wizard via setup:

```bash
datzi setup --wizard
```
