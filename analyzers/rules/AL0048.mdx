## Description

Use `Guard.NotNegative()` instead of verbose negative-check and throw patterns.

## Bad Code

```csharp
public void SetCount(int count)
{
    if (count < 0)
        throw new ArgumentOutOfRangeException(nameof(count));
}
```

## Good Code

```csharp
public void SetCount(int count)
{
    Guard.NotNegative(count);
}
```
