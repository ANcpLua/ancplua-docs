## Description

The `[Traced]` attribute requires a non-empty `ActivitySourceName` to identify which `ActivitySource` the generated spans belong to.

## Bad Code

```csharp
[Traced("")] // empty name
public partial class OrderService { }
```

## Good Code

```csharp
[Traced("MyService.Orders")]
public partial class OrderService { }
```
