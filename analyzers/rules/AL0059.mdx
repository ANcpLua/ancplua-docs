## Description

`typeof(T)` returns a globally shared `Type` object. Locking on it creates cross-assembly deadlock risks.

## Bad Code

```csharp
public void Update()
{
    lock (typeof(MyService)) { _count++; }
}
```

## Good Code

```csharp
private static readonly object s_lock = new();
public void Update()
{
    lock (s_lock) { _count++; }
}
```
