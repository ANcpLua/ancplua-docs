## Description

Use `Guard.NotZero()` instead of verbose zero-check and throw patterns.

## Bad Code

```csharp
public void Divide(int divisor)
{
    if (divisor == 0)
        throw new ArgumentException("Cannot be zero", nameof(divisor));
}
```

## Good Code

```csharp
public void Divide(int divisor)
{
    Guard.NotZero(divisor);
}
```
