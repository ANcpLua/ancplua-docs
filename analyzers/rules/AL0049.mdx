## Description

Use `Guard.Positive()` instead of verbose positive-check and throw patterns.

## Bad Code

```csharp
public void SetPageSize(int size)
{
    if (size <= 0)
        throw new ArgumentOutOfRangeException(nameof(size));
}
```

## Good Code

```csharp
public void SetPageSize(int size)
{
    Guard.Positive(size);
}
```
