## Description

Use `Guard.NotEmpty()` instead of verbose empty-GUID check and throw patterns.

## Bad Code

```csharp
public void Load(Guid id)
{
    if (id == Guid.Empty)
        throw new ArgumentException("GUID cannot be empty", nameof(id));
}
```

## Good Code

```csharp
public void Load(Guid id)
{
    Guard.NotEmpty(id);
}
```
