## Description

Interned strings are shared across the entire AppDomain. Locking on a string literal can deadlock with unrelated code.

## Bad Code

```csharp
public void Update()
{
    lock ("myLock") { _count++; }
}
```

## Good Code

```csharp
private readonly Lock _syncRoot = new();
public void Update()
{
    lock (_syncRoot) { _count++; }
}
```
